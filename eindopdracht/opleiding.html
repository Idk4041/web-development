over mij Projects Opleiding

School

Ik volg de opleiding Software Developer op het ROC in Almere Buiten. Ik heb voor deze opleiding gekozen omdat ik al langere tijd geïnteresseerd ben in computers, websites en hoe dingen online gemaakt worden. Ik vind het leuk om te leren hoe je een website vanaf de basis opbouwt en hoe je met HTML, CSS en later ook JavaScript, SQL en PHP iets kunt maken dat echt werkt en met servers kan praten.

Opleiding

Tijdens de opleiding Software Developer op het ROC leer ik hoe je websites maakt met HTML, CSS en JavaScript. Ik heb al verschillende dingen geleerd, zoals werken met code, het gebruiken van GitHub en het plannen van opdrachten met een projectboard. Ook leer ik steeds beter hoe ik problemen moet oplossen in mijn code, en hoe ik samen kan werken met andere studenten aan een project via Git en GitHub.

Stage

In het tweede leerjaar ga ik een stageplek zoeken die bij mij past en waar ik veel plezier mee kan hebben. Ik heb nog geen zekere plek gevonden, maar ik ken iemand die een bij een softwarebedrijf werkt en mij graag zou willen helpen met een stageplek.
